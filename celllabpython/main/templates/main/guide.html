Главная страница

  • фагоцит - питается нитратами в супе, тоесть разбросанными кусочками еды(не может питатся под жирной оболочкой)
  • Фотоцит - Питается светом супа
  • Флагеллоцит\шгутоцит - приводит механизм в движение жгутиками, может быть настроен сигналами.
  • девороцит - Ворует еду у клеток к которым прикасается и убивает если высосет всю еду, от него защищает кератиноцит, остеоцит
  • липоцит - Запасает в себе питательные вещества в виде липидов, полезная клетка для хранения еды и яиц
  • Кератиноцит - защищает соседние клетки и себя от девороцитов вироцитов и всех вредящих клеток
  • буэцит - может тонуть подниматся при плотности
  • глюоцит\Не знаю как на русском наверное клейкоцит - присасывается к клеткам к которым касается, работает для стенок
  • вироцит - клетка вируса при прикосновении превращает соседнюю клетку в клетку которую вы ему укажете
  • нитроцит - Вырабатывает вещество для деления(Без этого вещества организм не может делится при определёных настройках супа)
  • Стереоцит - может выводить сигналы по типу слева и справа, используется для управления
  • Сенсеоцит - говорит есть ли вокруг него определённые вещества но не указывает направление
  • миоцит - клетка мышцы, может быть настроенна сигналами. Используется для поворота организма сжимания разжимания адгезина перемещения по суше. нейроцит - как нейроны в мозгу может принимать сигналы и отдавать, делать колебания
  • секроцит - может вырабатывать яды, снимать жир с нитратов тоесть еды которые станут после этого сьедобными для фагоцитов. Используются для снарядов яда, запутывания жертвы т.к там можно сделать запах еды света и тд
  • стемоцит\стволоцит - принимает сигналы\свой возраст\массу\вещество для деления адгезиновые соединения, при определённых значениях данного сигнала может делится на указанную клетку
  • Гамета - как яйцеклетка и сперматозоид, при совмещении смешивает гены и образуют новый организм все гены двух оргов перемешиваются
  • Цилиоциты - могут вращать клетки которые соприкасаются с ними
  • Остеоцит - кератиноцит на минималках со своей фишкой. Он не защищает соседние клетки но защищает себя. Так-же это клетка кости т.к её адгезин твёрже.
